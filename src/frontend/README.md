# EmphaTech Frontend (Vite + React + Tailwind)

Run locally:

```powershell
cd src/frontend
npm install
npm run dev
```

API base URL used by the frontend: http://localhost:3000/api (configurable via REACT_APP_API_URL)

Pages included:
- /login
- / (dashboard)
- /send
- /recharge
- /transactions

This is a minimal scaffold to get you started. Next steps: style to match Opay, add signup/auth endpoints, and refine API contract with your backend.
